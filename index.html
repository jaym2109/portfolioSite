I'm Jason MacCuaig.

I'm a Technical Accountant based out of London, ON, Canada.

What is a Technical Accountant?

Accountants primarly deal with forecasting, budgeting and close-outs, all while providing KPIs and analytics to help key stakeholders gauge performance. I provide all these valuable services and much more.

As my career has progress I've discovered a new found passions for programming, development, business intelligence and pretty much all things tech. This has lead me to performing more technical tasks in the workforce such as; automating workflows, building financial servers and web applications, extracting important data from SQL and noSQL databases, working with the latest BI programs, system implementations, and many other technical tasks.

What Can I Do For You?

I come with a wide variety of services in both Technical and tradional Accounting & Financial skill sets.

Data Analytics

Data analytics is about more then just building reports for stakeholders. It's about telling a story, finding out what your data means, and providing proper insight into your business operations.

Corporate Accounting

By trade I am an accountant. I've worked in every possible scope in the corporate accounting world. From functional accounting (AP, AR, Payroll, etc.) to Strategic Planning I've done it all.

Learn More

Financial Modelling

Whether doing cost-benefit analysis, financial forecasting, or strategic planning. Corporations need to understanding the numbers and plan for the future. As an analyst it is my duty to interpret the information and develop models to plan for future endevours.

Learn More

Programming

I'm fluent in javascript and python programming languages. I often use programming to automate some of my more tediuous workflows including moving data between databases, sending out reminders to key stakeholders of upcoming deadlines and transforming data between excel spreadsheets.

Learn More

Business Intelligence

As an analyst you become accustom to dealing with data. I develop BI models using ETL principles to extract data from a variety of data sources to develop dynamic BI dashboards.

I've worked with some of the latest BI software tools including Microsoft Power BI and Tableau.

Learn More

Web Development

I build full-stack applications using Node.js, MongoDB, React and Express. I specialize in building micro applications that compliment corporate ERP systems. My biggest project to-date is a financial server that hosts financial statements online for stakeholders to access anywhere in the world.
